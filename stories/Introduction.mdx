# Introduction

Welcome to the Rubik's Cube Storybook!

This Storybook provides an interactive environment for developing, testing, and documenting UI components in isolation.
It helps ensure components are robust, accessible, and easy to use.

## About This Project

This project is a Rubik's Cube application built with React.
It includes features such as a timer, cube visualization, and user interaction tools.

## Running Storybook

To start Storybook locally, run the following command in your project root:

```bash
npm run storybook
```

This will open Storybook in your browser for interactive development.

## Adding New Stories

- New stories should be placed next the to underlying components.
- Use `.stories.tsx`, or `.mdx` files to document and demonstrate your components.
- For shared or global stories, use the `stories/` directory.

Refer to existing stories for examples and best practices in story creation.
