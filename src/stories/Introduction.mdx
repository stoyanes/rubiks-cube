# Introduction

Welcome to the Rubik's Cube Storybook!

This Storybook provides an interactive environment for developing, testing, and documenting UI components in isolation.
It helps ensure components are robust, accessible, and easy to use.

## About This Project

This project is a Rubik's Cube application built with React.
It includes features such as a timer, cube visualization, and user interaction tools.

## How to Use Storybook

- Browse the sidebar to explore available components and features.
- View live demos and documentation for each component.
- Use the controls panel to interact with component props.
